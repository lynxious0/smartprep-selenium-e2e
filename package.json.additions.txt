Notes on logging (new):
- Every run now writes a plain-text log via e2e/selenium/reporters/file-logger.reporter.js.
- e2e/selenium/logs/latest-<browser>.log — full detail of the most
  recent run for that browser: timestamp, PASS/FAIL/SKIP per test,
  duration, the test's remark (see below), and the assertion error for
  any failure. Overwritten each run.
- e2e/selenium/logs/history.log — the same detail, appended across
  every run/browser, so nothing is lost when "latest" gets overwritten.
- e2e/selenium/logs/last-combined-summary.log — written by
  `test:e2e:all`, a one-line-per-browser pass/fail summary.
- Every test in the suite calls `t.diagnostic('Remark: ...')` as its
  first line — a plain-English note on what that test is actually
  verifying and why. These remarks are captured in the log next to
  that test's result.
